# 버튼

&lt;Button /&gt; 컴포넌트 사용법 정리

<br />
<br />

## 타입

- 버튼의 형태를 type으로 지정해 줄 수 있습니다.
- type = `primary` | `default` | `dashed` | `text`

<Canvas>
  <Story name="type">
    <Button type="primary">Normal</Button>
    <Button type="default">Normal</Button>
    <Button type="dashed">Normal</Button>
    <Button type="text">Normal</Button>
  </Story>
</Canvas>

<br />

## 비활성화

- 버튼을 비활성화 해줄 수 있습니다.
- disabled가 true 일 때,

<Canvas>
  <Story name="disabled">
    <Button type="primary" disabled>Dimmed</Button>
    <Button type="default" disabled>Dimmed</Button>
    <Button type="dashed" disabled>Dimmed</Button>
    <Button type="text" disabled>Dimmed</Button>
  </Story>
</Canvas>

<br />

## 크기

- 버튼의 크기를 지정할 수 있습니다.
- size = `large` | `middle` | `small`

<Canvas>
  <Story name="size">
    <Flex gap=""></Flex>
    <Row><Col></Col></Row>
    <Button type="primary" size="large">Normal</Button>
    <Button type="primary" size="middle">Normal</Button>
    <Button type="primary" size="small">Normal</Button>
  </Story>
</Canvas>

<br />

## 풀사이즈
- 너비 100%를 갖는 버튼을 만들 수 있습니다.
- fullSize가 true 일 때,

<Canvas>
  <Story name="fullSize">
    <Button fullSize type="primary">FullSize</Button>
  </Story>
</Canvas>

<br />

## 뷰모드

- 뷰 모드 버튼으로 전환할 수 있습니다.
- viewMode가 true 일 때,

<Canvas>
  <Story name="viewMode">
    <Button viewMode>Normal</Button>
  </Story>
</Canvas>
