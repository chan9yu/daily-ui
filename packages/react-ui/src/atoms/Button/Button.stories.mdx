# Button

## type (color)

- button의 color를 **type** 속성으로 지정할 수 있습니다.
- default 값은 primary (#29aaff) 입니다.
- type: `primary` | `secondary` | `success` | `error` | `gray` | `white`

## size

- button의 크기를 **size** 속성으로 지정할 수 있습니다.
- default 값은 base 입니다.
- size: `sm` | `base` | `lg` 

<Canvas>
  <Story name="Size">
    <Button size="sm">Normal</Button>
    <Button size="base">Normal</Button>

  </Story>

## radius

- button의 테두리 둥글기를 radius 속성으로 지정할 수 있습니다.
- default 값은 sm 입니다.
- size: `sm` | `base` | `lg` | `xl` | `xxl`
	- sm: 4px
	- base: 6px
	- lg: 8px
	- xl: 16px
	- xxl: 32px
