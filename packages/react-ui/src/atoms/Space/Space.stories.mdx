# Space

&lt;Space /&gt; 컴포넌트 사용법 정리
- Space는 구성요소 간격(여백)을 설정할 때 유용합니다.
- Space 컴포넌트에 해당 구성요소를 children로 넘겨 사용합니다.

<br />

## side

- Space의 적용할 방향을 side 속성으로 지정할 수 있습니다.
- default 값은 all (모든 방향 적용) 입니다.
- side: `all` | `top` | `right` | `bottom` | `left` | `x` | `y`
	- top: 해당 구성요소에 위 (padding 혹은 margin) 여백을 설정
	- right: 해당 구성요소에 오른쪽 (padding 혹은 margin) 여백을 설정
	- bottom: 해당 구성요소에 아래 (padding 혹은 margin) 여백을 설정
	- left: 해당 구성요소에 왼쪽 (padding 혹은 margin) 여백을 설정
	- x: 해당 구성요소에 위 (padding 혹은 margin) 여백을 설정
	- y: 해당 구성요소에 위 (padding 혹은 margin) 여백을 설정

## space

- Space의 (여백) 값을 space 속성으로 지정할 수 있습니다.
- default 값은 xs (12px) 입니다.
- space: `xxxs` | `xxs` | `xs` | `sm` | `lg` | `xl` | `xxl` | `xxxl`
	- xxxs: 4px
	- xxs: 8px
	- xs: 12px
	- sm: 16px
	- md: 24px
	- lg: 32px
	- xl: 48px
	- xxl: 72px
	- xxxl: 96px
