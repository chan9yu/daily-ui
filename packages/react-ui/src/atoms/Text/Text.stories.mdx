# Text

&lt;Text /&gt; 컴포넌트 사용법 정리

<br />

## size

- Text의 사이즈를 **size** 속성으로 지정할 수 있습니다.
- default 값은 base (1rem = 16px) 입니다.
- size: `xs` | `sm` | `base` | `lg` | `xl`
	- xs: **0.75rem = 12px**
	- sm: **0.875rem = 14px**
	- base: **1rem = 16px**
	- lg: **1.125rem = 18px**
	- xl: **1.25rem = 20px**

	</Story>

## type (color)

- Text의 color를 **type** 속성으로 지정할 수 있습니다.
- default 값은 text (#2c3137) 입니다.
- type: `text` | `description` | `input` | `guide` | `white` | `primary` | `secondary` | `success` | `error`

## weight

- Text의 두께를 **weight** 값으로 지정할 수 있습니다.
- default 값은 regular (400) 입니다.
- weight = `thin` | `light` | `regular` | `medium` | `bold`
	- thin: **100**
	- light: **300**
	- regular: **400**
	- medium: **500**
	- bold: **700**
