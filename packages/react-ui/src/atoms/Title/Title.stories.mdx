# Title

&lt;Title /&gt; 컴포넌트 사용법 정리

<br />

## size

- title의 사이즈를 **size** 속성으로 지정할 수 있습니다.
- default 값은 2 입니다.
- size의 값은 h-tag에도 영향을 줍니다. (size = {2} 일 때, &lt;h2&gt;...&lt;/ h2&gt;)
- size: `1` | `2` | `3` | `4` | `5` | `6`;
	- 1: **2.5rem = 40px** / &lt;h1&gt;Title-1&lt;/ h1&gt;
	- 2: **2rem = 32px** / &lt;h2&gt;Title-2&lt;/ h2&gt;
	- 3: **1.75rem = 28px** / &lt;h3&gt;Title-3&lt;/ h3&gt;
	- 4: **1.5rem = 24px** / &lt;h4&gt;Title-4&lt;/ h4&gt;
	- 5: **1.25rem = 20px** / &lt;h5&gt;Title-5&lt;/ h5&gt;
	- 6: **1rem = 16px** / &lt;h6&gt;Title-6&lt;/ h6&gt;

## type (color)

- title의 color를 **type** 속성으로 지정할 수 있습니다.
- default 값은 title (#2c3137) 입니다.
- type: `title` | `description` | `input` | `guide` | `white` | `primary` | `secondary` | `success` | `error`

## weight

- title의 두께를 **weight** 값으로 지정할 수 있습니다.
- default 값은 regular (400) 입니다.
- weight = `thin` | `light` | `regular` | `medium` | `bold`
	- thin: **100**
	- light: **300**
	- regular: **400**
	- medium: **500**
	- bold: **700**
